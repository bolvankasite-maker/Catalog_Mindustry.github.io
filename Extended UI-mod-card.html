Extended UI

Мод для игровой индустрии, расширяющий
пользовательский интерфейс полезными функциями.

Описание

Мод добавляет много удобных Функций!
Он был создан 11 августа 2023 года

Что добавляет мод?

  • Пользовательский интерфейс
    Основная часть модификации. Работает только на вашем клиенте. Большинство функций здесь включены по умолчанию.

    - Таблица схем

    Быстрый доступ к вашим схемам. Разделены на категории (первый ряд кнопок).
    Настраивается — щелкните правой кнопкой мыши (дважды коснитесь на мобильном)
    1. На этикетке указано название категории.
    2. Нажмите на кнопку категории (первая строка), чтобы выбрать иконку.
    3. Нажмите на кнопку со схематической иконкой(все остальные),
    4. чтобы установить иконку или схему.Вы также можете изменить количество
    строк/столбцов или размер кнопок в настройках мода.

    - Таблица единиц измерения

    Отображает до 6 типов юнитов, которые есть у 8 лучших команд на карте, отсортированных по относительной ценности.
    Также есть несколько дополнительных функций:

    1. Наведите курсор на спрайт юнита на столе, чтобы увидеть,
    где он находится и на каком расстоянии.
    2. Нажмите на спрайт юнита на столе,
    чтобы отслеживать юнитов этого типа.

    - Силовая планка

    Показывает общее состояние питания.
    Также предупреждает о разделении сети

    - Тестирование (по умолчанию отключено)

    1.Показывает эффективность зданий для тестирования схем (WIP)
    2.Показывает, какой процессор управляет модулями для тестирования логики

    - Норма использования ресурсов

    Показывает, сколько ресурсов вы производите

    - Другое

    Расширяет границы минимального и максимального масштабирования
    Сохранение схем размером более 35x35. Работает в режиме наблюдателя
    Добывайте песок и другие нерудные полезные ископаемые
    Индикатор выполнения на заводе

    1. Полосы здоровья и щита юнита (требуют много кадров в секунду)
    2. Показывает дополнительную информацию о вражеских блоках
    3. Отслеживает курсор других игроков
    4.Улучшенный конвейер/система подачи


  • Взаимодействовать

    Дополнительная часть модификации.
    Автоматически взаимодействует с API игры
    (поэтому вас могут забанитьза их использование, будьте осторожны)
    Для всех взаимодействий используется общий таймер,
    который можно настроить в параметрах (по умолчанию 500 мс).
    начения ниже 250 мс часто приводят к временной блокировке серверов,
    поэтому я не рекомендую их использовать.
    По умолчанию полностью отключено.

    - Билдинги с автоматическим заполнением

    Автоматически размещает ресурсы с вашего дрона в зданиях, которые:

    1. Можете принять их
    2. Можно использовать их (например, генераторы, но не контейнеры)

    Если Allow interact with core automatically включен:

    1. Забирает ресурсы у ядер, если они нужны ближайшим зданиям
    2. Возвращает ресурсы в ядро, если они не подходят

    Всё довольно просто: для генераторов выбирается первый
    подходящий ресурс (например, уголь для сжигания),
    для турелей — лучшие боеприпасы, а со всем остальным и так понятно

Требования

Для корректной работы мода требуется:

  • Не указано, для какой версии игры Mindustry можно скачать Extended UI.
    Известно, что мод работает только на клиенте.
  • Минимум 2 ГБ оперативной памяти
  • С улыбкой скачивать мод (без улыбки не скачиваем)

Как скачать мод?


1. Клкните на кнопку Скачать мод
2. При открытие ссылки нажмите на зелёную кнопку код / code
3. В небольшом окошке нажимаем на кнопку скачать файл / download ZIP
4. Заходим в игру, во вкладке "модификации" нажать на "Импортировать модификацию",
найти только что скачанный файл, нажать на него - начать играть с модом
